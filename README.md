# CareerNet

Project setup
* `git remote add careernet https://github.com/huangshan108/CareerNet.git`
* `git pull`
* `bundle install`
* `rake db:migrate`
* `rails server`
Then open web browser, go to `localhost:3000`

TO commit to this repo
* `git add <file-name>`
* `git commit -m <message>`
* `git push careernet <branch>`
